# Debug Laravel

Objetivo: encontrar la causa leyendo lo minimo.

Reglas:

- No recorrer todo el proyecto.
- Identificar primero mensaje exacto, archivo, linea y modulo.
- Usar `rg` antes de abrir archivos completos.
- Responder con `Causa`, `Archivo` y `Arreglo` si aplica.
